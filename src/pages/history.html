1950
🖥️
Era Mainframe
Primele SO pentru calculatoare uriașe. Batch processing, carduri perforate, camere întregi.
1964
🎯
IBM System/360
Primul SO cu multiprogramming. Mai multe job-uri rulau simultan. O revoluție!
1969
🔓
Nașterea Unix
Bell Labs creează Unix. Design modular, portabil. Baza pentru toate SO-urile moderne.
1981
💻
MS-DOS
Microsoft DOS pe IBM PC. Calculatoarele personale intră în case și birouri.
1984
🎨
Apple Macintosh
Primul GUI pentru mase. Click în loc de comenzi. Revoluția interfeței utilizator.
1985
🪟
Windows 1.0
Prima încercare Microsoft de GUI. Modest început pentru ce urma să domine.
1991
🐧
Nașterea Linux
Linus Torvalds creează Linux. Open source, gratuit, puternic. Viitorul serverelor.
1995
⭐
Windows 95
Punctul de cotitură. Start button, taskbar. Vândut în milioane. Dominația începe.
2001
🎯
XP & OS X
Windows XP stabil și popular. Mac OS X unește frumusețea cu puterea Unix.
2007
📱
Revoluția Mobilă
iPhone lansează iOS. Android urmează. Miliarde au acum SO în buzunar.
2015+
🚀
Era Modernă
Windows 10/11, Cloud, IoT, AI. SO-urile sunt peste tot. Viitorul e aici.

  • Capitolul 1: Începuturile sistemelor de operare Primele calculatoare, primele interfețe, GM-NAA I/O
  • Capitolul 2: Era Mainframe Batch processing, carduri perforate, multiprogramare
  • Capitolul 3: Nașterea Unix Bell Labs, portabilitate, influența asupra SO moderne
  • Capitolul 4: Era PC și GUI MS-DOS, Macintosh, Windows, interfața grafică
  • Capitolul 5: Linux și Open Source Linus Torvalds, mișcarea open source, servere
  • Capitolul 6: Era Modernă Windows, macOS, Linux, mobil, cloud, AI
