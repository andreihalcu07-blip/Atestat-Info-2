Timeline: Evoluția Sistemelor de Operare

1950s-1960s

Primele Sisteme de Operare

UNIVAC, IBM System/360 OS

Primele SO au fost dezvoltate pentru calculatoarele mainframe. Acestea executau o singură sarcină la un moment dat. Utilizatorii trebuiau să folosească carduri perforate pentru intrare.

1970s

Era Unix și Multiprogramării

Unix (1969)

Unix a adus o revoluție prin portabilitate și design modular. A permis mai mulți utilizatori să lucreze simultan pe același calculator. Limbajul C a făcut Unix să se răspândească ușor.

1980s

Calculatoare Personale - DOS și Macintosh

MS-DOS (1981), Mac OS (1984)

Microsoft DOS a dominat calculatoarele personale. Apple Macintosh a introdus interfața grafică cu utilizatorul (GUI). Era calculatoarelor personale a început să înlocuiască mainframeurile.

1990s

Windows și Linux

Windows 3.0+ (1990), Linux (1991)

Windows 3.0 a făcut GUI accesibil pentru masele. Linux a apărut ca alternativă open-source. Internetul a început să devină dominant. Windows 95 a marcat o schimbare majoră în interfața SO.

2000s

Era Cloud și Mobile

Windows XP (2001), Mac OS X (2001), Android (2008), iOS (2007)

Apariția sistemelor de operare mobile. Cloud computing a început să se dezvolte. Seria Windows XP a fost extrem de populară. Consolidarea Linux pe servere.

2010s-2026

Era Modernă - AI și Hibrid

Windows 10/11, macOS, Linux, iOS, Android

Sistemele de operare devin mai inteligente cu AI. Computere hibrid și fără limită între device-uri. Creșterea importanței securității și privacy. Open-source devine mainstream.

Tipuri Majore de Sisteme de Operare

Windows

Cel mai utilizat SO pe desktop-uri. Dezvoltat de Microsoft. Domină piața calculatoarelor personale cu o cotă de piață de aproximativ 75%.

  • Versiuni: Windows XP, Vista, 7, 8, 10, 11
  • Arhitectură: Monolitic
  • Licență: Proprietară

macOS

Sistemul de operare al calculatoarelor Apple. Cunoscut pentru stabilitate și design elegant. Bază Unix, foarte popular printre creativi și developeri.

  • Versiuni: Yosemite, Sierra, Big Sur, Monterey, Ventura
  • Arhitectură: Hibrid (kernel XNU)
  • Licență: Proprietară

Linux

Sistemul de operare open-source, gratuit. Domină serverele web cu peste 96% din market share. Foarte flexibil și customizabil.

  • Distribuții: Ubuntu, Debian, Red Hat, CentOS
  • Arhitectură: Monolitic
  • Licență: GPL (open-source)

Android

SO pentru dispozitive mobile. Dezvoltat de Google. Cel mai utilizat SO la nivel global cu peste 70% din piață de smartphone-uri.

  • Bază: Linux kernel
  • Versiuni: Gingerbread până la Android 14+
  • Licență: Open source (parțial)

iOS

SO exclusiv pentru iPhone și iPad. Dezvoltat de Apple. Cunoscut pentru securitate și ecosistem închis.

  • Versiuni: iOS 1.0 până la iOS 17+
  • Arhitectură: Hibrid (kernel XNU)
  • Licență: Proprietară

ChromeOS

Sistemul de operare al Chromebook-urilor. Ușor, rapid și securizat. Bazat pe Linux, focusat pe cloud computing.

  • Bază: Linux kernel
  • Interfață: Chrome browser
  • Licență: Open source
