Resurse și Lecturi

Află mai mult despre sistemele de operare

📚 Cărți Recomandate

  • "Operating Systems: Three Easy Pieces"

    Una dintre cele mai bune cărți pentru înțelegerea conceptelor SO. Disponibilă gratuit online.

  • "Modern Operating Systems"

    O carte clasică și comprehensivă despre SO-uri, cu exemple din lumea reală.

  • "The Linux Programming Interface"

    Profund și detaliat, pentru cei care doresc să înțeleagă Linux la nivel de programare.

🎓 Cursuri Online

  • MIT OpenCourseWare - 6.828 Operating System Engineering

    Cursul MIT gratuit, oferit de una din cele mai bune universități din lume.

  • Coursera - Operating Systems Design and Implementation

    Cursuri interactive cu certificare opțională.

  • Udacity - Operating Systems Nanodegree

    Program intensiv pentru dezvoltatori care doresc să se specialize în SO.

🌐 Site-uri Informative

  • The Linux Foundation

    Organizația care suportă Linux. Informații, programe de training, și comunitate.

  • Wikipedia - Operating Systems

    Informații detaliate și referințe către sute de SO-uri, istorii și comparații.

  • OS.js

    Vizualizare interactivă a cum funcționează sistemele de operare moderne.

💻 Documentație Oficială

  • Microsoft Learn - Windows Operating System

    Resurse oficiale de la Microsoft pentru înțelegerea Windows.

  • Apple Developer Documentation - macOS

    Documentație completă pentru dezvoltatorii care lucreaza cu macOS.

  • Linux Man Pages

    Documentația completă pentru fiecare comandă și funcție Linux.

🎬 Video Educative

  • YouTube - Crash Course Computer Science

    Seri de videoclipuri educative care explică conceptele SO într-un mod ușor de înțeles.

  • YouTube - TechTalk Channels

    Preleceri și discuții despre evoluția și design-ul SO-urilor.

📦 Instrumentele și Software

  • VirtualBox

    Software gratuit pentru a rula mai mulți SO-uri pe același calculator. Perfect pentru a experimenta.

  • QEMU

    Emulator puternic pentru a testa diferite SO-uri și arhitecturi.

  • GDB (GNU Debugger)

    Instrument pentru debugging la nivel low, util pentru înțelegerea SO-urilor.
