Despre Windows 10

Windows 10 a unit cel mai bun din Windows 7 și Windows 8. A readus Start Menu-ul clasic combinat cu Live Tiles și a introdus Cortana, Edge și Windows Hello.

Start Menu Reimagined

🎙️

Cortana

Asistent vocal digital integrat în Windows pentru prima dată.

🌐

Microsoft Edge

Browser complet nou, înlocuiește Internet Explorer. Reconstruit pe Chromium în 2020.

🖥️

Virtual Desktops

Desktop-uri virtuale native pentru organizare mai bună.

📱

Continuum

Comută automat între moduri desktop și tabletă.

Windows Hello

Autentificare biometrică: amprentă, recunoaștere facială, iris.

Action Center

Centru de notificări și setări rapide unificate.

🎮

Xbox Integration

Xbox app, Game Bar, streaming de pe Xbox, DirectX 12.
