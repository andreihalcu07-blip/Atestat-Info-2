Windows 11

A new era for the PC

📅 26 Ianuarie 2026 💙 Fluent Design 2.0
🪟

Design Complet Nou

Interfață modernă cu colțuri rotunjite, efecte Mica, taskbar centrat și un nou Start Menu fără Live Tiles.

📱

Android Apps

Prima versiune Windows care rulează aplicații Android nativ prin Windows Subsystem for Android (WSA).

🎮

Gaming Enhanced

DirectStorage, Auto HDR, Xbox Game Pass integration pentru cea mai bună experiență gaming.

🤖

Copilot AI

Asistent AI integrat în Windows 11, accesibil din taskbar pentru ajutor instant.

💡 Did you know?

Windows 11 sare de la Windows 10 deoarece Microsoft a sărit numărul 9 (Windows 9 nu a existat niciodată din cauza problemelor de compatibilitate cu cod care verifica "Windows 9*" pentru Windows 95/98).

💻 System Requirements

⚠️
Cerințe Stricte

Windows 11 are cerințe hardware semnificativ mai stricte decât Windows 10, necesitând TPM 2.0 și Secure Boot.

🔧
Processor 1 GHz, 2+ cores, 64-bit
💾
RAM Minimum 4 GB
💿
Storage 64 GB sau mai mult
🔐
TPM TPM 2.0 (obligatoriu)
🛡️
Secure Boot UEFI, Secure Boot capable
🖼️
Display 720p, 9" diagonal, 8 bpc

✨ New Features

🪟

Snap Layouts & Groups

Organizează rapid ferestrele în layout-uri predefinite. Grupurile salvează configurația pentru taskbar.

📰

Widgets

Panou personalizat cu știri, vreme, calendar, stocks și alte informații la un click distanță.

💬

Teams Integration

Microsoft Teams integrat în taskbar pentru comunicare instantă cu prieteni și colegi.

🖥️

Virtual Desktops Enhanced

Fiecare desktop virtual poate avea propriul wallpaper și set de aplicații deschise.

🤖

Copilot

Asistent AI bazat pe tehnologie GPT, integrat direct în Windows pentru productivitate.

📁

Tabbed File Explorer

În sfârșit! Tab-uri în File Explorer pentru navigare mai eficientă.

🔊

New Sounds

Sunete noi, mai calme și moderne pentru notificări și acțiuni sistem.

⚡

Performance

Optimizări pentru RAM, CPU și baterie. Foreground windows primesc prioritate.

🎨 Fluent Design 2.0

Windows 11 introduce a doua iterație majoră a sistemului de design Fluent, cu accent pe simplitate, calm și conexiune emoțională.

Rounded

Rounded Corners

Toate elementele UI au colțuri rotunjite pentru un aspect mai prietenos și modern.

Mica

Mica Material

Efect de transparență care preia culorile din wallpaper pentru o experiență coerentă.

Acrylic

Acrylic

Efect de sticlă mată pentru meniuri și flyouts, creând adâncime.

Centered Taskbar
