Microsoft Windows 95

Versiune 4.0 | 24 August 1995

Windows 95 a fost un sistem de operare revoluționar lansat de Microsoft, care a schimbat fundamental modul în care oamenii interacționează cu calculatoarele personale.

A introdus pentru prima dată butonul Start, taskbar-ul și Windows Explorer - elemente care sunt prezente și astăzi în Windows modern.

Noutăți în Windows 95

  • 🚀 Meniul Start - Acces rapid la programe și setări
  • 📊 Taskbar - Bara de activități pentru multitasking
  • 📁 Windows Explorer - Manager de fișiere modern
  • 🖱️ Click dreapta - Meniuri contextuale
  • 📛 Nume lungi de fișiere - Până la 255 caractere
  • 🔌 Plug and Play - Instalare automată hardware
  • 🌐 TCP/IP integrat - Suport nativ pentru Internet
  • 🎮 DirectX - Platformă pentru jocuri
